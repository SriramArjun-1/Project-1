# Dolphin's Colive — Static Website

This repository contains a small static website for Dolphin's Colive PG (paying-guest accommodation).

Quick start

1. Open `index.html` in your browser, or run a local server from the project folder:

```bash
cd Project-1
python -m http.server 8000
# then open http://localhost:8000
```

What's included

- `index.html` — main page with home, rooms, gallery, testimonials and contact form
- `css/styles.css` — styles and responsive layout
- `js/script.js` — simple interactivity: responsive menu, smooth scroll and form handling
- `assets/` — small SVG logo, favicon and image placeholders

Next steps
